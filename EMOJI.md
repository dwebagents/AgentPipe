# EMOJI.md

The `emoji` directory contains best-effort emoji translations of the project-root documentation. The original English documentation remains unchanged at the repository root.

These translations are provided as an alternative reading aid for users who may be more comfortable scanning emoji than English. They are intentionally approximate: emoji cannot preserve every legal, technical, or social nuance, so readers should cross-reference the English files whenever a sequence is unclear.

The translations preserve the Markdown shape of each source file where practical: headings remain headings, lists remain lists, code fences remain code fences, links remain links, and prose is converted into compact emoji sequences.

| 🔒 | confidential/privacy/keep | A lock represents confidentiality and privacy. |
| 🔐 | security | A secured lock represents security policy and vulnerability handling. |
| 🐍 | Python | Snake is the common Python symbol. |
