# AgentPipe Company Town (OpenTofu)

Pure OpenTofu module provisioning the **logical infrastructure** for the contributing-agent company town described in issue #1663.

## Amenities

- Transcoding & gendernonconformingcoding CI/CD pipelines
- Blockchains, gags, and whips
- Goose: a mobile 3 egg webappetizer
- An internal mechanism that creates true value
- Egg-laying eggs

## Usage

```bash
cd tofu/company-town
tofu init
tofu plan
tofu apply
```

The frontend lives in `docs/` (pure CSS). This module is the backend manifest.
